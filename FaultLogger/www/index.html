Registration

When registering, ensure that you insert a valid student number and set a password.
Your student number will become your username when logging in.

Login

Before attempting to login, ensure you have registered.
Once registered, set your password and use your valid student number as your username.

Data Capturing

When capturing the data, simply fill out the form and submit it.
If the "type" of social gathering that you are capturing is not available,
you can add a new type via the "New" link.

Watch list Function

This function allows for the user to create a watch list for areas.
Simply change add the area you wish to keep an eye on...

Location

There are two options for capturing the locational data:

Option 1: GPS
This option sets the location to your phone's GPS co-ordinates.

Option 2: Manual Selection
This option allows for the manual selection of the location.
Simply tap on the mobile screen to drop a pin and set the location.
